## How to start

```cp .env.example .env.local```

Update the value for the .env file accordingly

``npm i``

``npm run dev``

## How to use it

Considering you have the backend running this minimal frontend will only display the ID of the nefturian for a given wallet (or the information if the Nefturian has been minted).

It will also show the general distribution of the 1240 Nefturians.
